tutorialr.com Tutorial Tuesday - Windows App SDK - Set Eighteen

12th March 2024

It is the last Tutorial Tuesday for now and tutorialr.com has been updated with four new Tutorials for Windows App SDK that you can follow to create some custom looking controls.

Custom RadioButton
Custom RatingControl
Custom TextBox
Custom ComboBox

Customised Controls include Custom RadioButton which applied a custom look to a RadioButton, Custom RatingControl which customises the look of a RatingControl, Custom TextBox which changes the appearance of a TextBox and Custom ComboBox for a custom looking ComboBox.

There will be more Tutorials for Windows App SDK coming later this year, which will include mini-apps to create your own small applications along with more complex examples to help understand and implement a variety of functionality in Windows App SDK which will eventually complete the set of one-hundred Tutorials!

RoguePlanetoid Weekly Update #47

10th March 2024

This week was an interesting one as went along to Tech on the Tyne to see Paul Cheetham talk about Using emotional data to make optimal decisions which I also wrote an Article about. Emotional intelligence wasn't something I consciously thought about but have been doing so ever since the talk, even referencing it a couple of times since, so suspect that talk will have a long-term impact on how I think about how I feel about things and how I relate to how others feel about something, you never know what you may learn by going along to an event and am pleased I went along to learn about something different that I can apply from now on.

This week I've also started writing the next Episode of the RoguePlanetoid Podcast about .NET MAUI which I covered in my recent talk, Take a Note of XAML with .NET 8, continuing the theme started in this month's Episode about XAML where I covered current uses of XAML with Windows Presentation Foundation and Windows App SDK along with legacy platforms that used XAML too. .NET MAUI is something I only started to use when putting together the demo, however, Xamarin which .NET MAUI is based on is something I used a few years ago and am pleased the progress it has made and something I'll make use of in the future!

Next week will see the final four Tutorials for Windows App SDK being published for now on tutorialr.com, originally this was to allow for the InkCanvas control to be added in the next version of Windows App SDK due out in six months, however that may be delayed to the next version of Windows App SDK. I'll still be publishing more Tutorials at the end of the year and use an alternative solution until that feature is added.

RoguePlanetoid Weekly Update #46

3rd March 2024

This week was packed full of events starting with North East Dev Community which was held at Durham University at their Department of Computer Science and Department of Mathematical Sciences, it was a fantastic venue! The event started with Andrew Buckingham talking about Collections, Classes and Clocks, which was then followed by my own talk, Take a Note of XAML with .NET 8, which I really enjoyed delivering and was the first time I had my own clicker for the slides which came in handy. I also wrote an Article about the North East Dev Community - February 2024 event which covered both talks!

This week also saw yet another Silicon Mingle, it was amazing to go along to the Town Wall pub for this great networking event which had a few familiar faces from recent events I'd only met there along with some new faces plus those I'd met at previous events! It was fantastic to catch up with those I'd met before and met new people there, it is a fantastic event to see what people are doing and how they can and do help each other, there's no feeling of competition here in the North East just a feeling of collaboration and is great to be part of that culture!

Yesterday saw DDD North held at the University of Hull, and I travelled down on the Friday to Hull from Gateshead via Newcastle and Leeds and stayed just across the road from the venue so would not have far to go in the morning! It was great to see a few familiar faces there too and meet up with more people, the first session was Introduction to StereoKit Mixed Reality Development by Lee Englestone followed by Supercharge your data with Azure AI Search and Open AI by Pete Gallagher then onto May the Forms Be with You: A New Hope with Blazor Hybrid on WinForms by Don Wibier before breaking for Lunch. The first session after lunch was my own, an extended version of Take a Note of XAML with .NET 8, and it went well and had great questions and interest in the talk which was fantastic! The day ended with Extending Copilot with Copilot Studio by Dylan Hayes who announced he had just been recently made a Microsoft MVP, which was great! The day ended with a prize draw and although didn't win anything it was great to see those who did including one who won a Concorde Lego set that I had my eye on complementing me on my talk which was great and congratulated them on winning the set. The only downside of the day was missing talks that coincided with ones I had been to and hope to catch those in some form in the future! Overall DDD North was a great community event to be part of that I always look forward to attending or speaking at and wrote an Article covering the sessions I attended, including my own on DDD North 2024.

RoguePlanetoid Podcast - Episode Fourteen - XAML

1st March 2024

Today sees the release of Episode Fourteen of the RoguePlanetoid Podcast about XAML, XAML or eXtensible Application Markup Language allows developers to build applications that separate a rich user interface from the implementation including events, business objects and other code.
