tutorialr.com Tutorial Tuesday - Windows App SDK - Set Eleven

7th February 2023

It is another Tutorial Tuesday and tutorialr.com has been updated with four new Tutorials for Windows App SDK that you can follow and build some fun games!

Order Game
Slide Game
Codes Game
Match Game

Games include Order Game where the aim of the game is to place all the squares in the correct order in the quickest time, Slide Game is based upon those fifteen-piece sliding puzzle games where you need to get the squares in the correct order. There's also Codes Game where you need to correctly guess the four-digit code and Match Game where you need to remember a pattern then reproduce it as many times as you possibly can!
