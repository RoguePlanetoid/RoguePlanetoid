RoguePlanetoid Weekly Update #38

7th January 2024

It is a new year and a new start and a new day for the RoguePlanetoid Weekly Update! There hasn't been one for a couple of months but aim to publish a weekly update every Sunday throughout 2024. Firstly, it was great to publish the first RoguePlanetoid Podcast episode of 2024 with an Overview of 2023, last year was the busiest year ever and already have a feeling that this year will be even busier and appreciate every person who has either read these Posts or Articles on the website or has listened to the Podcast.

Secondly, this week saw the first Tutorial Tuesday for tutorialr.com with two brand-new Windows App SDK Tutorials for a Gauge Control and Clock Control, every two weeks from now on I will be posting four brand-new tutorials including user controls and customised versions of existing controls for Windows App SDK. The aim is to get to one hundred tutorials by the end of the year, some of which rely on some up-coming features so may need to take a break to make sure can use those features when they're available.

Finally, I'll be doing a talk for the North East Dev Community Meetup where I'll be talking about ways you can use XAML with .NET including .NET MAUI, Windows App SDK and Windows Presentation Foundation with my talk "Take a note of XAML with .NET 8", I'm looking forward to delivering my first in-person talk of the year, and was a great excuse to try something with .NET MAUI for the first time. In putting together that demo I've realised some new tutorials on .NET MAUI would be useful so may work on that later this year! Otherwise, I'm looking forward to seeing what 2024 brings for events, talks and more and will post about everything I'm working on or doing right here!
