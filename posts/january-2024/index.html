tutorialr.com Tutorial Tuesday - Windows App SDK - Set Fifteen

30th January 2024

It is another Tutorial Tuesday and tutorialr.com has been updated with four new Tutorials for Windows App SDK that you can follow to build some useful user controls.

Ruler Control
Segment Control
Matrix Control
Split Control

Controls include Ruler Control which can be used to display an on-screen ruler in metric and imperial. There is also a a Segment Control for a seven-segment display, Matrix Control for a dot-matrix display and Split Control for a split-flap display all of which can be used to display the date and or time plus any supported value.
