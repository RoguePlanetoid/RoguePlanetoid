RoguePlanetoid Weekly Update #66

25th August 2024

This week has been a really busy week with networking event by North East Network and Platform which I covered in my Article about it at Platform Newcastle - August 2024 with some great insights and a show and tell with all presentations being really interesting and glad this time it was just me listening but I did really enjoy last month's one where I got to present about how we aim to help people get started in podcasting or help them on their podcasting journey with Cluarantonn

RoguePlanetoid Weekly Update #65

18th August 2024

These past couple of weeks have been busy with some more new networking events to go along to such as the North East Network Pop Along event and another studio tour for Cluarantonn including speaking to other businesses as part of the PLMS group we recently joined which has been great to see how those in the group can help and how we can help them and there will be more conversations, tours and networking next week too!

These last couple of weeks have been busy with working on a new presentation, Write Windows 11 Widgets with Windows App SDK, which is something I've been doing for the past few weeks alongside the other things I was doing these past few weeks! Although not yet announced there are some updated Packages on NuGet that I've published including Comentsys.Toolkit with a 1.1.0 version which adds a helpful Class for Data Binding as well as some other small updates. There is also a new 2.0.0. version of Comentsys.Toolkit.WindowsAppSdk updated for .NET 8 and addition of helper Classes developed by Microsoft for Widgets in Windows 11.

Next week there will be more things to do but also to continue with the new presentation, Write Windows 11 Widgets with Windows App SDK. It is good to not only work on ways to help podcasters get started or help those who have a podcast already but continue creating as a software developer, I'd never done Widgets before doing the short Talk on tutorialr.com about Windows App SDK with .NET. However now I've learned more about them and made it easier for other developers to get started and the next Episode of the RoguePlanetoid Podcast in September will be about Windows 11 Widgets!
