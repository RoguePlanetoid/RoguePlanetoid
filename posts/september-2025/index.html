RoguePlanetoid Weekly Update #86

8th September 2025

Last week saw the release of the latest Episode of the RoguePlanetoid Podcast about NuGet which is the go-to resource for packages for .NET developers to leverage the functionality there or contribute their own. Popular packages on NuGet allow developers to focus on their core functionality while leveraging tried and tested libraries or if there's useful functionality to share with others then can publish your own packages. I've created Toolkits, Asset packages and more that I have published on NuGet which you can find via comentsys.com/packages.

Last week the latest Epsiode of the Podcast was a timely topic as I published my latest package of Comentsys.Toolkit.Blazor which is a set of components for Blazor Server and / or Blazor WebAssembly which has components including one to show Assets from my asset packages along with Clock, Dial, Directional Pad, Directional Stick, Donut, Gauge, SegmentDisplay, MatrixDisplay and Sector. It was fun to put this together with the aid of GitHub Copilot and it has been amazing to see that Comentsys.Toolkit.Blazor has already gained over 120 downloads.

This month I'm pleased to say that I will be on-stage hosting a panel discussion at Tech Connect 2025: The Future of AI , where we'll talk about some of the topics discussed during the day and more! I'm honoured to be part of what I'm sure will be an amazing event held in The Catalyst in Newcastle upon Tyne on Tuesday 30th September so if you want to come along then sign up for your ticket today and catch me along with a whole host of other amazing people for the conference later this month!

Comentsys.Toolkit.Blazor 1.0.0 Released

2nd September 2025

Today sees the Release of version 1.0.0 of my Toolkits package of Comentsys.Toolkit.Blazor for Blazor Server and / or Blazor WebAssembly on both NuGet and open-source on GitHub.
