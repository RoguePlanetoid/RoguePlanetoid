Comentsys Assets - September 2024

6th September 2024

Comentsys.Assets.FluentIcons

Comentsys.Assets.FluentIcons is a newly available free open-source Asset Resource for the Regular Fluent UI System Icons targeting .NET Standard 2.0 with a Initial Release of 1.0.0 in September 2024. Comentsys.Assets.FluentIcons uses SVG or Scalable Vector Graphics for the Regular style of 2,599 Fluent UI System Icons from GitHub created by Microsoft.

Comentsys.Assets.FluentIcons supports replacing the colour used for Regular Fluent UI System Icons as needed in a .NET Standard 2.0 compatible project such as Blazor and Windows App SDK. Comentsys.Assets.FluentIcons source code is available on GitHub at github.com/Comentsys/Comentsys.Assets.FluentIcons or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.FluentIcons.

Comentsys.Assets.FluentEmoji

Comentsys.Assets.FluentEmoji is a free open-source Asset Resource for Flat Fluent Emoji targeting .NET Standard 2.0 with an Initial Release of 1.0.0 in September 2022 and has been updated in September 2024 with 1.1.0 for Comentsys.Toolkit version 1.1.0 along with fixes to a few Emoji. Comentsys.Assets.FluentEmoji uses SVG or Scalable Vector Graphics for the Flat style of Fluent Emoji from GitHub created by Microsoft.

Comentsys.Assets.FluentEmoji supports replacing the colours used for the Emoji that enables developers to differentiate the same Emoji with different styles in their .NET Standard 2.0 compatible projects such as Blazor and Windows App SDK. Comentsys.Assets.FluentEmoji source code is available on GitHub at github.com/Comentsys/Comentsys.Assets.FluentEmoji or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.FluentEmoji.

Comentsys.Assets.FluentEmoji.Shaded uses SVG or Scalable Vector Graphics for the Colour style of Fluent Emoji from GitHub created by Microsoft.

Comentsys.Assets.FluentEmoji.Shaded supports replacing the colours used for the Emoji so that enables developers to differentiate the same Emoji with different styles in their .NET Standard 2.0 compatible projects such as Blazor and Windows App SDK. Comentsys.Assets.FluentEmoji.Shaded source code is available on GitHub at github.com/Comentsys/Comentsys.Assets.FluentEmoji.Shaded or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.FluentEmoji.Shaded.

Comentsys.Assets.Flags

Comentsys.Assets.Flags is a free open-source Asset Resource of Flags from all countries in the world targeting .NET Standard 2.0 with an Initial Release of 1.0.0 in September 2022 and has been updated in September 2024 with 1.1.0 for Comentsys Toolkit version 1.1.0. Comentsys.Assets.Flags uses SVG or Scalable Vector Graphics from Flag Icons for 266 world flags on GitHub created by Lipis and curated by Koppi.

Comentsys.Assets.Flags supports Flags as Rectangles or Squares but does not allow for colour customisation to allow the Flags to remain as intended for use in .NET Standard 2.0 compatible projects such as Blazor or Windows App SDK. Comentsys.Assets.Flags source code is available on GitHub at github.com/Comentsys/Comentsys.Assets.Flags or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.Flags.

Comentsys.Assets.Games

Comentsys.Assets.Games is a free open-source Asset Resource for Games targeting .NET Standard 2.0 with an Initial Release of 1.0.0 in September 2022 and has been updated in September 2024 with 1.1.0 for Comentsys.Toolkit version 1.1.0. Comentsys.Assets.Games uses SVG or Scalable Vector Graphics from Wikimedia including full set of six black or white Chess Pieces created by Cburnett and a full set of forty-four Mahjong tiles created by Shizhao.

Comentsys.Assets.Games supports replacing the colours used for Asset Resource for Chess and Mahjong allowing developers to create colourful experiences for Games targeting .NET Standard 2.0 compatible projects such as Blazor or Windows App SDK. Comentsys.Assets.Flags source code is available on GitHub at github.com/Comentsys/Comentsys.Assets.Games or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.Games.
