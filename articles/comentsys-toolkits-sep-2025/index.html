Comentsys Toolkits - September 2025

2nd September 2025

Comentsys.Toolkit.Blazor (1.0.0)

Comentsys.Toolkit.Blazor 1.0.0 is a free open-source toolkit with Components for Blazor Server and Blazor WebAssembly released in September 2025. Components include Asset to display an Asset Resource from any Comentsys.Asset.* package in Blazor Server or Blazor WebAssembly and Clock for an analogue clock for Blazor Server that can be real-time in Blazor WebAssembly. Components for Blazor WebAssembly only include Dial along with Directional Pad and Directional Stick for games. Displays such as SegmentDisplay for a seven-segment style display and MatrixDisplay for a five-by-seven matrix style display for Blazor Server inclding show date / time and can be real-time in Blazor WebAssembly. Finally there are Components for Blazor Server and Blazor WebAssembly including Donut to display donut / pie charts, Gauge for an analogue gauge and Sector to for sectors used by the Component of Donut.

Comentsys.Toolkit.Blazor offers a variety of Components that can work with Blazor Server along with ones that work with Blazor WebAssembly plus Components that support both Blazor Server and Blazor WebAssembly. Documentation for Comentsys.Toolkit.Blazor includes examples of the Components which were ported over from Comentsys.Toolkit.WindowsAppSdk or other examples to create an interesting set of Components for developers to use in Blazor applications or even games.

Comentsys.Toolkit.Blazor can be installed into any Blazor application and this package supports from .NET 8 to the latest .NET 9. Comentsys.Toolkit.Blazor source code and documentation is available on GitHub at github.com/Comentsys/Comentsys.Toolkit.Blazor or get details of the NuGet package at nuget.org/packages/Comentsys.Toolkit.Blazor.
