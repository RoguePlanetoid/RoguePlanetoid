Comentsys Assets - August 2025

20th August 2025

Comentsys.Assets.FluentIcons (2.0.0)

Comentsys.Assets.FluentIcons 2.0.0 is the free .NET Standard 2.0 based open-source Asset Resource for Regular Fluent System Icons initially released in September 2024, previously updated in October 2024 and now updated again in August 2025. The RegularFluentIcons have been updated with new additions as well as fixes from the Microsoft Fluent UI System Icons Repository on GitHub. Comentsys.Assets.FluentIcons now contains Metadata with FluentIconMeta which exposes FluentIconData for each Icon which includes name, keywords and direction indicating use with left-to-right and right-to-left languages.

Comentsys.Assets.FluentIcons is a useful way of being able to use an array of Fluent System Icons in your .NET applications not only as SVG data but access the raw Path data for icons for use with compatible UI technologies such as XAML. There is also updated documentation which includes an example image showing every single Fluent System Icon.

Comentsys.Assets.FluentIcons with over 340 downloads since launch can be installed into any .NET Standard 2.0 compatible project such as Windows App SDK, .NET MAUI, Blazor or ASP.NET Core. This package supports from .NET Core 2.0 all the way to the latest .NET 9 for modern applications along with .NET Framework 4.6.1 to .NET Framework 4.8.1 for legacy applications. Comentsys.Assets.FluentIcons source code and documentation is available on GitHub at github.com/Comentsys.Assets.FluentIcons or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.FluentIcons.

Comentsys.Assets.FluentIcons.Filled (2.0.0)

Comentsys.Assets.FluentIcons.Filled 2.0.0 is the free .NET Standard 2.0 based open-source Asset Resource for Filled Fluent System Icons initially released in September 2024, previously updated in October 2024 and now updated again in August 2025. The FilledFluentIcons have been updated with new additions as well as fixes from the Microsoft Fluent UI System Icons Repository on GitHub. This package leverages the latest version of the Comentsys.Assets.FluentIcons package which includes Metadata for every single Icon including name and keywords.

Comentsys.Assets.FluentIcons.Filled offers the Filled style of Fluent System Icons to be used in your .NET applications and there is also updated documentation with an example showing every single icon that is supported by the package. It will be interesting to see how these icons will be used in .NET applications especially now there are even more to choose from!

Comentsys.Assets.FluentIcons.Filled with over 280 downloads since launch can be installed into any .NET Standard 2.0 compatible project such as Windows App SDK, .NET MAUI, Blazor or ASP.NET Core. This package supports from .NET Core 2.0 all the way to the latest .NET 9 for modern applications along with .NET Framework 4.6.1 to .NET Framework 4.8.1 for legacy applications. Comentsys.Assets.Flags (2.0.0)

Comentsys.Assets.Flags 2.0.0 is the free .NET Standard 2.0 based open-source Asset Resource for Flags for all Countries in the World initially released in September 2022, previously updated in September 2024 and now updated again in August 2025. It has been updated with additions as well as fixes to Flags from the Flag Icon Repository on GitHub. Comentsys.Assets.Flags now contains Metadata with FlagMeta which exposes FlagData for each Flag which includes the name and code for each Country along with Capital and Continent information where available.

Comentsys.Assets.Flags is a great way of being able to use square and rectangular flags of the world in your .NET applications easily. I decided not to allow the flag colours to be replaced like other Asset Resource packages but as a fun addition you can now get the flag colours which can be used for theming experiences. It is great to be able to leverage this resource which has been kindly updated and maintained and pleased am able to share it for .NET developers to be able to use.

Comentsys.Assets.Flags with over 480 downloads since launch can be installed into any .NET Standard 2.0 compatible project such as Windows App SDK, .NET MAUI, Blazor or ASP.NET Core. This package supports from .NET Core 2.0 all the way to the latest .NET 9 for modern applications along with .NET Framework 4.6.1 to .NET Framework 4.8.1 for legacy applications. Comentsys.Assets.Flags source code and documentation is available on GitHub at github.com/Comentsys/Comentsys.Assets.Flags or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.Flags.

Comentsys.Assets.Games (2.0.0)

Comentsys.Assets.Games 2.0.0 is the free .NET Standard 2.0 based open-source Asset Resource for Games including Chess, Mahjong, Playing Cards, Dice, Dominoes, Piece and Board initially released in September 2022, updated in September 2024 and now updated again in August 2025 with 2.0.0 with Playing Cards by TheRealRevK from commons.wikimedia.org/wiki/Category:SVG_playing_cards_6 along with original designs I've created for Dice and Dominoes plus Piece and Board which can be used to create a variety of games.

Comentsys.Assets.Games features my own designs for the Dice and Dominoes as well as the Piece which includes a square, circle, rounded and hexagon and Board which can be used together with existing Assets for Chess to create your own Chess game or other games such as Draughts. With elements of most other types of game available it should be possible for .NET developers to create a variety of games using them and will be interesting to see how they are used!

Comentsys.Assets.Games with over 140 downloads can be installed into any .NET Standard 2.0 compatible project such as Windows App SDK, .NET MAUI, Blazor or ASP.NET Core. This package supports from .NET Core 2.0 all the way to the latest .NET 9 for modern applications along with .NET Framework 4.6.1 to .NET Framework 4.8.1 for legacy applications. Comentsys.Assets.Games source code and documentation is available on GitHub at github.com/Comentsys/Comentsys.Assets.Games or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.Games.
