Comentsys Assets - July 2025

23rd July 2025

Comentsys.Assets.FluentEmoji (2.0.0)

Comentsys.Assets.FluentEmoji 2.00 is the free .NET Standard 2.0 based open-source toolkit for Flat Fluent Emoji initially released in September 2022, previously updated in September 2024 and now updated again in July 2025. The FlatFluentEmoji have been updated with new additions as well as fixes from the Microsoft Fluent Emoji Repository on GitHub. Comentsys.Assets.FluentEmoji now contains with Metadata with FluentEmojiMeta which exposes FluentEmojiData for each Emoji which includes links to an Image that can be used with an Image Resource of ImageFluentEmoji along with links for many Emoji that are Animated with a link that can be used with an Image Resource of AnimatedFluentEmoji. Other metadata includes glyph and Emoji representations as well as the name of the Emoji for use in applications.

Comentsys.Assets.FluentEmoji is a fun and fantastic way to inject a fantastic array of Fluent Emoji into your .NET applications, and not only can you use the existing SVG-based Flat style, but the PNG based 3D style which would be megabytes in size if included directly along with some Emoji supporting animated PNGs which would be gigabytes in size if referenced directly. There is also updated documentation which includes an example image showing the Flat, 3D and static version of the Animated Emoji available.

Comentsys.Assets.FluentEmoji with over 750 downloads since launch can be installed into any .NET Standard 2.0 compatible project such as Windows App SDK, .NET MAUI, Blazor or ASP.NET Core. This package supports from .NET Core 2.0 all the way to the latest .NET 9 for modern applications along with .NET Framework 4.6.1 to .NET Framework 4.8.1 for legacy applications. Comentsys.Assets.FluentEmoji source code and documentation is available on GitHub at github.com/Comentsys/Comentsys.Assets.FluentEmoji or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.FluentEmoji.

Comentsys.Assets.FluentEmoji.Shaded (2.0.0)

The ShadedFluentEmoji have been updated with new additions as well as fixes from the Microsoft Fluent Emoji Repository on GitHub. This package leverages the latest version of the Comentsys.Assets.FluentEmoji package which includes Metadata for every single Emoji along with links to PNGs for 3D image-based Emoji and animated Emoji where available.

Comentsys.Assets.FluentEmoji.Shaded offers the Colour style of Fluent Emoji that are highly detailed and interesting to use in your .NET applications. There is also updated documentation with an example showing every single emoji that is supported by the package. I've used earlier versions of this package in my own tutorials and examples and look forward to seeing how these amazing emoji designed by Microsoft can be leveraged in other applications.

Comentsys.Assets.Display (2.0.0)

Comentsys.Assets.Display 2.0.0 is the free .NET Standard 2.0 based open-source assets package for a Segment and Matrix Display initially released in September 2024 and now updated in July 2025 with 2.0.0 with new styles for the Matrix Display including Circle, Hexagon and Octagon which joins the existing Square style along with the Segment Display. Each display type of Matrix and Segment can be set to a custom colour or array of colours to produce interesting and eye-catching results.

Comentsys.Assets.Display Assets were designed by me with the Segment type created in InkScape for each part of the Seven-Segment Display. The Five-by-Seven Dot Matrix Display also was designed in InkScape by hand for the Squares and now with Circles, Hexagons and Octagons which were then processed for use in the package to allow them to be used in your .NET applications. I've already made use of the original version of the package and look forward to using it more in the future and seeing how others use it too. The documentation for the package has also been updated to include images of how the Segment and different styles of Matrix display would look with the different values supported.

Comentsys.Assets.Display with over 140 downloads can be installed into any .NET Standard 2.0 compatible project such as Windows App SDK, .NET MAUI, Blazor or ASP.NET Core. This package supports from .NET Core 2.0 all the way to the latest .NET 9 for modern applications along with .NET Framework 4.6.1 to .NET Framework 4.8.1 for legacy applications. Comentsys.Assets.Display source code and documentation is available on GitHub at github.com/Comentsys/Comentsys.Assets.Display or get details of the NuGet package at nuget.org/packages/Comentsys.Assets.Display.
