Spotify Codes

3rd February 2023

Spotify is a music streaming service offering music with many albums and artists to discover along with podcasts and audiobooks and there is a great way to share these with others using the Spotify app with Spotify Codes where you can enter a Spotify URI for the content you want to share with a Spotify Code from spotifycodes.com

Get Spotify URI

However, due to changes in the Spotify web experience and apps, you are not able to get a Spotify URI easily as you used to. So, first here is a guide on how to create a Spotify URI needed from a Spotify Link. In the Spotify app on desktop or web you can search for an artist, album or track along with a podcast, episode or audiobook and then select the "..." option and then select the option containing somthing similar to copy a link or from the app you can then use Share then Copy link and you'll get something like the following Spotify Links:

  • Artist
    open.spotify.com/artist/562Od3CffWedyz2BbeYWVn?si=5kee0_xpQHCgOSLrultTAQ
  • Album
    open.spotify.com/album/0a3YQpBnRzJzNktOjb6Dum?si=UR6Vw7dLTHm8wkiHn03exA
  • Track
    open.spotify.com/track/4dAgaMrchuSCRrol1AAthD?si=8bad762977314609
  • Podcast
    open.spotify.com/show/2emlLsByzU2krtfBgo5yQ9?si=7f598e7373c044df
  • Episode
    open.spotify.com/episode/0rfObbivydRYp02dTEWebr?si=2cb6a063b93b4123
  • Audiobook
    open.spotify.com/show/2WbzV1xR3Gah1VhdzQjHPW?si=8844529eb9e54b28

Spotify Links are not a Spotify URL, but they do contain the pieces you need. They always begin with spotify: followed by the type of item which for artist is artist, album is album, track is track, podcast is show and episode is episode and audiobook is show, then this is followed by another colon of : then the Base-62 ID for examples you would get the following Spotify URI:

  • Artist
    spotify:artist:562Od3CffWedyz2BbeYWVn
  • Album
    spotify:album:0a3YQpBnRzJzNktOjb6Dum
  • Track
    spotify:track:4dAgaMrchuSCRrol1AAthD
  • Podcast
    spotify:show:2emlLsByzU2krtfBgo5yQ9
  • Episode
    spotify:episode:0rfObbivydRYp02dTEWebr
  • Audiobook
    spotify:show:2WbzV1xR3Gah1VhdzQjHPW

Download Spotify Code

To download a Spotify Code, enter a Spotify URI on spotifycodes.com and then select the Get Spotify Code option and you should see the Spotify Code along with any details of the item from Spotify. You can choose the Background Colour from any colour along with either White or Black for the Bar Colour and you can choose the Size such as 640 along with the Format from JPEG, PNG and SVG. Then you can choose the option to Download, you can then use an image of a Spotify Code for example on any posts on social media or even put it on a T-Shirt to help promote your favourite album or podcast.

Get Spotify Code

Alternatively, a Spotify Code could be displayed directly such as on a website, this can be accomplished by using the URL that links to a Spotify Code for the examples the URL would be as follows:

  • Artist
    https://scannables.scdn.co/uri/plain/jpeg/000000/white/640/spotify:artist:562Od3CffWedyz2BbeYWVn
  • Album
    https://scannables.scdn.co/uri/plain/jpeg/000000/white/640/spotify:album:0a3YQpBnRzJzNktOjb6Dum
  • Track
    https://scannables.scdn.co/uri/plain/jpeg/000000/white/640/spotify:track:4dAgaMrchuSCRrol1AAthD
  • Podcast
    https://scannables.scdn.co/uri/plain/jpeg/000000/white/640/spotify:show:2emlLsByzU2krtfBgo5yQ9
  • Episode
    https://scannables.scdn.co/uri/plain/jpeg/000000/white/640/spotify:episode:0rfObbivydRYp02dTEWebr
  • Audiobook
    https://scannables.scdn.co/uri/plain/jpeg/000000/white/640/spotify:show:2WbzV1xR3Gah1VhdzQjHPW

The jpeg section in this URL can be changed to png or svg for the Format, this is followed by 000000 which is the hex-colour for the Bar Colour followed by either white or black for the Background Colour then there is the 640 which is the Size, the last element is the Spotify URI. For the examples you would get the following Spotify Codes:

  • Artist
  • Album
  • Track
  • Podcast
  • Episode
  • Audiobook

Scan Spotify Code

You can scan a Spotify Code from the Spotify app on iOS and Android by launching the app and then going to the Search option and then select the Camera icon on the Search Box then point the Camera of your smartphone at the Spotify Code and the app should then recognise the artist, album, track, podcast, episode or audiobook and display it within the Spotify app.
